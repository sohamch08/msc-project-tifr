\chapter{Exponential Sums over Finite Fields}
Exponential sums are important tools in number theory for solving problems involving integers. These sums have many applications in Analytic Number Theory.

In this chapter we will set up exponential sums for finite fields by special group homomorphisms called \emph{characters}. Since a finite field has two different inherent group structure; one is additive and the other is multiplicative we obtain two different characters. Exponential sums are formed by using the values of one or more characters and possibly combining them with weights or with other function values.
